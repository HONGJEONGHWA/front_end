C팀 일어일본학과 11학번 남동우

주제1)화장품 관심 정도

-화장품 사용.미스트,스킨,로션 사용
-양이 많고 보습력이 좋은 것이 가장 큰 이유이지만 일단 받은 것이라서 씀

주제2)화장품 구매시 고려 사항

-피부타입과 가격을 중요시 생각함
-누나한테 물어보거나 매장에서 추천받는 것을 씀
-인터넷 검색은 하지 않음
-선택시 트러블 생길까봐 고민
-가격은 비싼데 왜 이 가경 주고 사야하는지 의문

주제3)성분 관심 정도
-성분에 대해서 잘 몰라서 성분고려한 적 없음.보통 Test 후 구입
-화해 어플 처음 들어봄. 방문 할 의향 없음
-발암물질 아닌이상 그냥 씀
-알로에 광고가 가장 기억에 남음.하도 짐승젤 거려서

주제4)화장품 사용시 특이사항
-중간에 그만 쓴 적 있음. 건조해지고 시간이 지나면 기름져서
-다시 구매 할 때 끈적이지 않고 보습력 좋은 것을 고려
-팔자 주름과 목주름,까만 피부 색이 고민
-바디로션,썬크림으로 피부 관리 함.엑소 코피지 팩해서 성공
-동화적으로 설명해주는 것은 색달라서 재밌을 것 같다.방문할 것 같다
-일반적인 남자들이 그것만 보고 끌릴지 고민.아예 타겟을 여성으로
-화장품이 광범위해서 위험부담 크다

-기초로 숨 화이트라인 사용
-가장 잘 맞고 발효 성분이 좋음. 지인의 추천으로 사용

주제2)화장품 구매시 고려 사항
-성분을 자세히 보지는 않음
-가격보다 피부타입에 맞춤
-Test해보고 오프라인 매장에서 구입함
-트러블 때문에 구매시 고민

주제3)화장품 관심 정도
-성분 고려한적은 딱히 없음
-화해 다운 받은 적은 있으나 성분을 별로 중요시 생각하지 않아거 지움
-안심할 수 있는 제품 선택이 좋음
-성분에 관심 없는 사람은 사용 안할 듯

주제4)화장품 사용시 특이사항
-중도에 바꾼적 있음.트러블이 나서 한번 바꿈.후에 추천받아서 순한 라인으로 씀
-인중에 트러블이 많이 나서 자기전 연고 바르고 메티폼패드 붙임
-디자인은 맑고 깨끗한 느낌이 좋을 것 같다
-방문할 의향이 있다
-추천 기능이 있으면 좋을 듯
-남성 지향 디자인 만들기 추천
